What is CSS?

Cascading style sheets.

Three ways to add CSS

  • External style sheet - link to the style sheet in the head of the html
  • Interal style sheet - uses a style tag in the head
  • Inline CSS - add a style attribute directly within an element tag

Types of CSS Selectors

  • Tags: p, div, h1, etc. (least specific)
  • Classes: .article, .page-header, etc. (more specific, overrides tag styles)
  • ID: #project1header, #profilediv, etc. (specific to one thing on a page) (most specific, overrides all other styles)
